# System-Informations-console

System-Informations-console is a lightweight C++ console application for Windows that displays real-time system information.

## Features

- 🖥️ RAM Information
  - Total RAM
  - Available RAM
  - Memory usage (%)

- ⚙️ CPU Information
  - Number of processor cores

- 💾 Disk Information
  - Total disk capacity
  - Free disk space

- 🎮 GPU Information
  - Graphics adapter name (DXGI)

- 🔄 Automatic refresh every 5 seconds
